# Student Course Selection Dashboard (CQI)

A Clean Architecture web application with a **Node.js/Express Backend** and **React/Vite Frontend**.

## Project Structure

- `backend/`: Express API server
- `frontend/`: React + Vite application

## Getting Started

### Prerequisites
- [Bun](https://bun.sh/) (or Node.js)

### Installation

1.  **Backend**:
    ```bash
    cd backend
    bun install
    ```

2.  **Frontend**:
    ```bash
    cd frontend
    bun install
    ```

## Running the Application

You need to run both the backend and frontend servers.

1.  **Start Backend**:
    ```bash
    cd backend
    bun start
    ```
    Runs on `http://localhost:5000`

2.  **Start Frontend**:
    ```bash
    cd frontend
    bun run dev
    ```
    Runs on `http://localhost:5173`

## Features
-   Secure Login (simulated)
-   Course Selection by Branch and Type
-   Detailed Course Information (Outcomes, Mappings)
